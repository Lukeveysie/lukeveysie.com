How to install in Apple Mail
  1. 1Click Select for Apple Mail on any signature below — it highlights the rendered signature.
  2. 2Press ⌘ C to copy it.
  3. 3Open Mail → Settings → Signatures. Pick an account, click + to create a new signature.
  4. 4Click inside the right-hand preview panel, select all placeholder text, then press ⌘ V to paste.
  5. 5Done — the logo and formatting will appear. Set it as default for your account.
